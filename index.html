Welcome

May 8th, 2015

Hello! I will be using this site to collect sources and document my understanding on the various coding topics required for my CIT 261 class. Here's the exact phrasing as found in my course's syllabus:

In order to produce effective applications for mobile devices and to accomplish the outcomes for the course (see the Syllabus), a strong knowledge, understanding, and ability is required in many areas. In order to achieve this it is suggested that each student needs to know, understand, and be able to use the following tools and concepts:

  1. The software development process and some standard methodologies,
  2. Creative, pleasant, mobile user interface creation
  3. JavaScript objects
  4. DOM manipulation
  5. CSS3 transitions
  6. CSS3 animations
  7. Using local data storage
  8. AJAX interactions with remote services
  9. JSON data serialization

This list is not presented in any specific order and is not comprehensive of all that is needed to be successful in the mobile application industry, this course, nor the web application industry. The student will also need to identify other tools, topics, and/or abilities they may need, are interested in, and then know, understand, and use them effectively and well.

So there you have it. That is exactly what this site hopes to accomplish. Navigation ain't that pretty... but feel free to check out the contents of this site by selecting the links at the top of this page.

Thanks for visiting!
