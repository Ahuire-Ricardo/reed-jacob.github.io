What Are Objects?

The simplest explanation for what a JavaScript object is comes from w3schools:

In real life, a car is an object.

A car has properties like weight and color, and methods like start and stop:

Object Properties Methods

car.name = Fiat

car.model = 500

car.weight = 850kg

car.color = white
car.start()

car.drive()

car.brake()

car.stop()

All cars have the same properties, but the property values differ from car to car.

All cars have the same methods, but the methods are performed at different times.

JavaScript has a number of predefined objects. (Some can be found here) Additionally, we are free to make our own objects. In order to create an object, it must first initilized. Here's an example of creating an object using JavaScript: 


            // Here, we demo a prototype for an object titled "car". This first section is called a "constructor function".
            // It allows for us to assign values to the object's propeties.

            function Car(make, model, year, color) {
                this.make = make;
                this.model = model;
                this.year = year;
                this.color = color;
            }

            // New objects can be created that follow the constructor function.

            var myCar = new Car("Honda", "Civic", "2004", "black");
            var myWifesCar = new Car("Chevrolet", "Avalanche", "2005", "blue");

            // Here's an example of displaying this newly assigned information in an HTML format:

            document.getElementById("demo").innerHTML = "I own a " + myCar.color + " " + myCar.make + " " + myCar.model + "."
                + " He's a little bit older, he was made in " + myCar.year + "."
                + " My wife's car is a " + myWifesCar.color + " " + myWifesCar.make + " " + myWifesCar.model + "."
                + " She's got one year on my " +  myCar.model + "! Her's was made in " + myWifesCar.year + "";
