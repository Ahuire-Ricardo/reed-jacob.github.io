Local Storage

June 3rd, 2015

Local storage enables web applications to store data locally within the user's browser. The browser has an allotted amount of space that is dedicated to storing user data. This information includes, but is not limited to: user preference settings, application data and other more temporary forms of relevant user data. HTML5 local storage provides two objects for storing data in the client:

window.localStorage (Stores data with no expiration date)
window.sessionStorage (Stores data for a single session)
