CSS 3 Transitions and Transforms

May 20th, 2015

What is CSS?

CSS (Cascading Style Sheets) is a style sheet langauge that is, most often, used to change the style of web pages and documents. CSS3 is the latest standard for CSS. While this page's contents will not explore all that CSS3 has to offer, it will display a two uses known as "transitions" and "transforms".

CSS3 Transitions

Simply put, CSS3 transitions provide a way to control animation speed when changing a particular CSS property. Instead of clicking a button and having immediate changes, the change will take place over a period of time (be it a few seconds or even a few minutes). Let's take a look at an example that manipulates a few different properties of a box:

.box {
    border-style: solid;
    border-width: 1px;
    display: block;
    width: 100px;
    height: 100px;
    background-color: #64D489;
    -webkit-transition:width 2s, height 2s, background-color 2s;
    transition:width 2s, height 2s, background-color 2s;
    color: #FFFFFF;
    text-align: center;
}
.box:hover {
    background-color: #0053A6;
    width:200px;
    height:200px;
    color: #FFFFFF;
    text-align: center;
  }

Output:

Hover me

The transitions changed the color, the width and the height of our box. Pretty neat, eh? It is important to remember to specify the time in which a transition will take place. In this example, we specified that it would occur over the course of 2 seconds. If you don't set it or you set it to 0, the transition will occur instantaneously... which kind of defeats the purpose of these nifty transitions! For a comprehensive list on CSS transition properties, check w3.org.

CSS3 Transforms

CSS3 transforms allow for content displayed on HTML pages to be modified without actually making changes to the content. For example, one could rotate, skew, scale, and translate elements both in plane and in a 3d space. CSS transforms in 3d space is definitely a little more complicated and well beyond the scope of my course's requirements. For that reason, we'll be taking a look at simple transforms in a plane (2d).
