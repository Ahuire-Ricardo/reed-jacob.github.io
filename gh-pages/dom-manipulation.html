DOM (Document Object Model) Manipulation

May 15th, 2015

DOM manipulation refers to using JavaScript to access and manipulate HTML documents and objects. The Safari Developer Library puts it this way:

The primary function of the Document Object Model is to view, access, and change the structure of an HTML document separate from the content contained within it.

By taking advantage of the DOM, you can:
  • Rearrange sections of a document without altering their contents
  • Create and delete existing elements as discrete objects
  • Search and traverse the document tree and the subtrees of any element on the tree
  • Completely isolate the documentâ€™s structure from its contents
