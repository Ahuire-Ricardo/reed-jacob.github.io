DOM (Document Object Model) Manipulation

May 15th, 2015

DOM manipulation refers to using JavaScript to access and manipulate HTML documents and objects. The Safari Developer Library puts it this way, "the primary function of the Document Object Model is to view, access, and change the structure of an HTML document separate from the content contained within it." The beauty of HTML and JavaScript is that they are easily configurable and can be modified in realtime. Let's take a look at some examples.

The first example is how to simply modify a element's style with JavaScript. The element we will change will be the color of displayed text:




Color Me




          

var xx = document.getElementById("id13959");
var button1 = document.getElementById("b1");
var button2 = document.getElementById("b2");

button1.addEventListener("click", function () {xx.style.color="red";} , false);
button2.addEventListener("click", function () {xx.style.color="blue";} , false);
