AJAX

June 12th, 2015

What is "AJAX"?

AJAX is a cleaning produc- okay, just kidding. AJAX stands for "Asynchronous Javascript and XML". AJAX allows the user to exchange data with the server and update certain parts of a web page without actually needing to refresh to see the results. w3schools.com provides us with a very simple example of how this can be implemented:

Press the button below to change this text with AJAX.

What's The Point?

The main appeal behind AJAX is that it allows for the webpage to update asynchronously by exchanging small amounts of data with the server behind the scenes. In other words, this means that things can "change" without needing to refresh the entire page. Think of when you use Facebook, YouTube, or Gmail. As you select different objects and communicate with the server, you are able to change them without needing to be redirected to an entirely new page (think reading comments or having "predictive text" when searching for something - here's an example of how that works!). These uses are a little more advanced than what I currently know. I will keep this page updated as I learn more.
