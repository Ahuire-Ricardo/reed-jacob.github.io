JSON

May 15th, 2015

What is "JSON"?

JSON stands for JavaScript Object Notation. JSON is a syntax for serializing objects, arrays, numbers, strings, booleans, and null. JSON uses JavaScript syntax, but the JSON format is text only, just like XML. Unlike XML, JSON is shorter, can use arrays, and is quicker to read and write data. Here's an example of utilizing JSON to create JavaScript objects:

// JSON objects are written within curly braces. Objects can contain multiple values.
// Commas are used to separate assignments (name:value).
var info = '{"name":"Jacob Reed","street":"250 N. Rosemont Cir.","zip":"85206"}';

// This parses our variable, "obj", as a string in JSON.
var obj = JSON.parse(info);

// Here we print our object and its separate values (name, street, and zip).
document.getElementById("demo").innerHTML =
obj.name + "
" +
obj.street + "
" +
obj.zip;

If we wanted to store more information we could create a JSON array. For example:

var names = [
  {
  "firstName":"Jacob",
  "lastName":"Reed",
  },
  {
  "firstName":"Michael",
  "lastName":"Jordan",
  },
  {
  "firstName":"George",
  "lastName":"Washington"
  }
];

function printName1() {
  document.getElementById("names").innerHTML =
  names[0].firstName + " " + names[0].lastName;
  }
function printName2() {
  document.getElementById("names").innerHTML =
  names[1].firstName + " " + names[1].lastName;
  }
function printName3() {
  document.getElementById("names").innerHTML =
  names[2].firstName + " " + names[2].lastName;
  }
